# boniq - Advanced Portfolio System

A modern, full-stack portfolio system built with Vue 3, Express, and MySQL. Features a stunning glassmorphism UI, admin dashboard, and complete content management.

## Tech Stack

**Frontend:** Vue 3, Vite, Pinia, Vue Router, Tailwind CSS, Axios
**Backend:** Node.js, Express, Sequelize ORM, MySQL, JWT, bcrypt
**Features:** Glassmorphism UI, animated backgrounds, admin CRUD, file uploads, visit tracking

## Quick Start

```bash
# Backend
cd backend
npm install
npm run dev

# Frontend
cd frontend
npm install
npm run dev
```

Open **http://localhost:5173** in your browser.

**Default Admin:** username `goxriddle` or `admin@system` / password `gutuza.24@`
